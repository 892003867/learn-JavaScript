#### _01_解构  
1. 数组的解构  
*按照文档描述，数组的解构属于‘模式匹配’，个人白话翻译就是一对一的匹配方式。匹配的键就是数组的索引值index，进行两边匹配赋值。  
>解构混入  
```javascript 1.6  
// 4.解构混入  
    fn(() => {  
        let arr = () => {  
            return {"a": 1}  
        }  
        let obj = {  
            ...arr() // 可以将arr的返回对象或者值解构到obj对象中  
            /*  
            * 结果：{a: 1}  
            * */  
        }  
        console.log(obj)  
    })  
```
